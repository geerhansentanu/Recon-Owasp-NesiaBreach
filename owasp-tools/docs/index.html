Masukkan Web. Jalankan Recon Aman.

GitHub Pages tidak bisa menjalankan Python langsung, jadi tombol ini membuka GitHub Actions. Kamu cukup paste target ke form workflow, lalu download report HTML/JSON setelah selesai.

Tidak perlu token di halaman ini. Kamu hanya perlu login GitHub dan punya akses menjalankan Actions di repo.

[idle] Input target, klik tombol, lalu paste di form GitHub Actions.
01 Klik `Buka Actions` dari halaman ini.
02 Klik `Run workflow`, paste target website.
03 Download artifact report setelah workflow selesai.
Dibuat oleh NesiaBreach. Gunakan hanya untuk target yang sudah terotorisasi.
